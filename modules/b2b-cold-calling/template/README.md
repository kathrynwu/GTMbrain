# B2B Cold Calling Generator

This template turns the module playbook into a browser-based script generator.

## What it does

- captures segment, buyer, pain, and offer
- recommends `A`, `B`, or `C`
- generates a 15-second opener
- generates a 30-second opener
- generates qualification questions
- generates a value bridge
- generates common objections and responses
- generates a meeting close
- shows what metrics to track

## Run locally

```bash
cd GTMbrain/modules/b2b-cold-calling/template
python3 -m http.server 4322
```

Then open `http://localhost:4322`.

## Notes

- this is static and local-first
- no API key is required
- it is meant to help you draft and iterate, not replace real call review
