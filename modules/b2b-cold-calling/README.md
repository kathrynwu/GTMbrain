## Try it locally

From your machine:

```bash
cd GTMbrain/modules/b2b-cold-calling/template
python3 -m http.server 4322
```

Then open `http://localhost:4322`.

If you only want the raw playbook, read the markdown files directly. If you
want a fast way to generate and iterate scripts, use the template.
