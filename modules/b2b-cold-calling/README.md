# B2B Cold Calling

Capture B2B cold-call scripts, review criteria, and tactical calling
heuristics.

Use this when a software company is selling to another software company.

## Summary

- **Surface:** `Playbook`
- **Audience:** outbound operator, GTM lead, growth engineer
- **Primary action:** read the playbook
- **Status:** first release

  local SMBs

## Focus

- B2B script patterns
- buyer authority
- objection handling
- practical benchmarks
- meeting-booking discipline

## What this playbook covers

- the 3-step B2B cold-call framework
- a call review rubric
- first-touch, follow-up, and gatekeeper frameworks
- objection handling and talk-track references
- opener rules for the first 6 seconds
- qualification prompts
- close and calendar-invite discipline
- when to choose call-first, enrich-first, or email-first

## Source material
